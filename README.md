# cns-ssl-pages

## 项目简介

本仓库用于归档和发布一系列独立 HTML 页面。这些页面不针对任何特定域名或网站，而是作为通用页面资源的集合，便于访问、分发和长期保存。

## 目录结构

```
cns-ssl-pages/
├── pages/        # 存放所有 HTML 页面文件
├── assets/       # 页面引用的静态资源（CSS、JS、图片等）
└── README.md     # 本文件
```

- `pages/` 目录下每个 HTML 文件代表一个独立页面，命名清晰，便于直接访问。
- `assets/` 目录存放页面所需的公共资源，按类型或用途组织。
- 根目录下的其他文件（如配置文件、说明文档）用于仓库维护。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务，可直接在浏览器中打开。
- 页面内容可能包含演示、示例、说明或工具类信息，无商业或推广目的。
- 归档遵循“一页一文件”原则，不混杂不同功能在同一文件中。
- 页面内可能引用外部资源（如 CDN 库），但尽量使用本地或稳定来源。

## 使用方式

1. 直接访问本仓库的 GitHub Pages 地址（若已启用）即可浏览页面列表。
2. 克隆或下载仓库后，在本地打开 `pages/` 下的 HTML 文件即可查看。
3. 如需引用页面中的资源，建议使用相对路径。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会在提交信息中注明。
- 欢迎通过 Issues 提交建议或报告问题，但请注意本仓库不提供技术支持。
- 不接受直接向 `pages/` 目录推送与归档主题无关的内容。
- 仓库文件变更会保留历史记录，可回溯每次修改。
